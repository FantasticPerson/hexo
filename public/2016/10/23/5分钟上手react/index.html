5分钟上手react

2016-10-23
#web

git地址：react-starter

运行步骤：

  1. 安装node

  2. 进入项目根目录，运行
    npm install
    有可能这条命令运行会很慢，我建议先安装 cnpm
    npm install cnpm -g
    然后再运行
    cnpm install

  3. 运行
    npm run dev
    通过浏览器打开 http://localhost:3999/#/demoPage
    下面就是运行效果：

好了，上面就是怎么把项目跑起来的全过程！！！

index.js

onLeave：自然表示的就是离开这个路由之后做的事情
childRoutes：表示当前这个路由底下的子路由
